# Roadmap Переезд

Интерактивный дашборд для отслеживания миграции на целевую архитектуру.

## Структура

```
├── generate.py          ← скрипт генерации (Python 3 + openpyxl)
├── data_entry.xlsx      ← файл для ввода данных (открывать в Excel)
├── source/              ← исходные справочники (Confluence-экспорт)
└── docs/
    └── index.html       ← дашборд (открывать в браузере)
```

## Как пользоваться

### 1. Редактирование данных
Откройте `data_entry.xlsx` в Excel:
- Столбец **Активен** — выберите «Да» для нужных комбинаций
- **Голубые столбцы** — плановые даты этапов
- **Зелёные столбцы** — фактические даты

### 2. Генерация дашборда
```bash
python3 generate.py
```

### 3. Просмотр
Откройте `docs/index.html` в браузере или перейдите по ссылке GitHub Pages.

## Установка

```bash
pip install openpyxl
```

## Этапы миграции

1. **Старт разработки** — начало работ
2. **Интеграционное тестирование** — e2e-тесты
3. **1%** → **5%** → **50%** → **100%** — поэтапная раскатка
